To run in command line:

1. type: 'make all'
2. Open a second command window.
3. Type: './server arg' , where arg is the port.
4. In the other window, type: './client localhost arg2' , where arg2 is the port from above.
5. To end the program, press 'ctrl+c' for the server, 'ctrl+z' for the client.

Note: to test filling the queue, press 'ctrl+c'.
